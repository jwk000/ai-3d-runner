## 当前玩法摘要

> 玩家在一个会向前延伸的四面方形隧道里自动奔跑。左右键默认切道；在边缘道且已起跳时，左右键会触发整条隧道旋转 90 度，让相邻墙面翻成新的地面。世界中只保留地面空洞这一种危险，目标是尽可能跑得更远。
